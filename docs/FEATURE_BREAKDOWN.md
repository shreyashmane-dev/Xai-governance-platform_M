# Feature Breakdown

## 1. Model Registry
- Upload and validate `.pkl`
- Metadata + checksum storage
- Version-aware listing

## 2. Dataset Registry
- Upload and validate `.csv`
- Top-10 row preview
- Schema profile storage

## 3. Metrics Engine
- Accuracy, Precision, Recall, F1, AUC
- Confusion matrix
- Stored per model-dataset pair

## 4. Explainability
- SHAP global importance
- API summary endpoint for assistant context

## 5. Governance
- Fairness score
- Demographic parity / equal opportunity differentials
- Subgroup analysis
- Trust score + risk class

## 6. Drift Monitor
- PSI per numeric feature
- Alert count and alert list

## 7. Reports
- Snapshot generation with checksum
- Governance-ready export source payload

## 8. AI Assistant
- Context-grounded prompt template
- OpenAI proxy + fallback response rules
- Chat logging with latency

## 9. Onboarding Tour
- React Joyride first-time detection
- Restart tour control from sidebar
