# User Manual

## Login
- Open `/auth` and sign in with Firebase credentials.

## Upload Model
- Go to `Models`.
- Upload `.pkl` model.
- Select active model from dropdown.

## Upload Dataset
- In `Models`, upload `.csv` dataset.
- Inspect top 10-row preview and choose dataset.

## Run Metrics
- Click `Run Metrics`.
- Review results in Dashboard.

## Explainability
- Open `Explainability`.
- Click `Run SHAP`.
- Review global feature importance chart.

## Governance
- Open `Governance`.
- Optionally enter sensitive column.
- Run scan and inspect trust score/risk/subgroups.

## Drift
- Open `Drift Monitor`.
- Select baseline/current datasets.
- Run drift analysis and review alerts.

## Reports
- Open `Reports`.
- Generate report snapshot for current model/dataset.

## Assistant
- Open `AI Assistant`.
- Ask trust/explainability/governance/drift questions.
- Assistant uses current context from platform state.
