# Demo Script (15-20 Minutes)

## 1. Platform Intro (2 min)
- Explain trust/governance objective.
- Show dashboard shell and modules.

## 2. Ingestion (3 min)
- Upload model `.pkl`.
- Upload dataset `.csv`.
- Show dataset preview.

## 3. Metrics + Explainability (4 min)
- Run metrics.
- Open dashboard chart.
- Run SHAP and inspect top features.

## 4. Governance + Drift (4 min)
- Run governance scan with sensitive column.
- Show trust score and subgroup table.
- Run drift between baseline/current datasets.

## 5. Assistant + Reports (4 min)
- Ask AI assistant why trust changed.
- Generate report snapshot.
- Open report list with checksum.

## 6. Close (2 min)
- Show auditability value and deployment readiness.
- Mention next phase roadmap.
